# Spend Wise 💸

Spend Wise is a full-stack personal finance tracker built using:

- React (Vite)
- Node.js
- Express
- MongoDB

It allows users to:

- Create and manage expenses
- Split expenses between multiple people
- Upload receipt images
- View spending reports and insights
- Track monthly budget usage


## Features

- Expense tracking
- Split expense calculation
- Receipt upload
- Budget alerts
- Reports dashboard
